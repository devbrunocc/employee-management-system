# Employee Management System 👥

Sistema desenvolvido em Java para gestão de folha de pagamento e cadastro de colaboradores, com foco em manipulação de coleções dinâmicas e expressões Lambda.

## 📌 Funcionalidades
* **Cadastro Dinâmico:** Permite registrar N funcionários conforme a necessidade do usuário.
* **Validação de Unicidade:** Verificação inteligente para impedir o cadastro de IDs duplicados.
* **Busca e Atualização:** Localiza um funcionário específico pelo ID para aplicação de reajuste salarial percentual.
* **Listagem Consolidada:** Exibição formatada de todos os funcionários registrados no sistema utilizando o método `toString()`.

## 🛠️ Conceitos Técnicos Aplicados
Neste projeto, consolidei conhecimentos fundamentais de Java moderno:
* **Listas (ArrayList):** Armazenamento dinâmico de objetos em memória, permitindo flexibilidade no tamanho da coleção.
* **Expressões Lambda & Streams:** Uso de `.stream().filter()` para realizar buscas de forma performática e legível.
* **Encapsulamento:** Proteção dos dados sensíveis (como o salário) utilizando modificadores de acesso `private` e métodos `get/set`.
* **Lógica de Controle:** Uso de loops `for` e `while` para garantir que o fluxo de entrada de dados seja robusto e livre de erros.
